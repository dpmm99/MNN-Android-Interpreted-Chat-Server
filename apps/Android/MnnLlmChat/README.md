To work on MNN Chat on my WSL Debian distribution that I set up:

export ANDROID_HOME=/usr/lib/android-sdk
export ANDROID_NDK=$ANDROID_HOME/ndk/27.3.13750724
export PATH=$PATH:$ANDROID_HOME/platform-tools
export PATH=$PATH:$ANDROID_HOME/cmdline-tools/latest/bin
cd /mnt/c/DePro/CodeProjects/NotMine/MNN-Android-Interpreted-Chat-Server/apps/Android/MnnLlmChat
sudo ./gradlew assembleStandardDebug

Then on Windows, use this to move it to the emulator:
cd C:\DePro\CodeProjects\NotMine\MNN-Android-Interpreted-Chat-Server\apps\Android\MnnLlmChat
adb -s emulator-5554 uninstall com.aureuscode.mnnchatserver
adb -s emulator-5554 install -r app/build/outputs/apk/standard/debug/app-standard-debug.apk

And just use this to put it on my phone when no emulator is running:
adb install app/build/outputs/apk/standard/debug/app-standard-debug.apk

To export an MNN model, given where I already have MNN and adb set up, go into WSL2 (owner/model would be something like "google/translategemma-4b-it"):
cd ~/MNN/transformers/llm/export
source .env/bin/activate
python3 llmexport.py --export mnn --path OwnerOnHuggingFace/ModelOnHuggingFace
(Optionally, append --quant_bit 8 for 8-bit quantization instead of 4-bit)
Copy the output model folder to the Windows side for convenience
Rename it to match the model and change NewModelFolderName in the command below to match
Execute: adb shell mkdir -p /data/local/tmp/mnn_models && adb push NewModelFolderName /data/local/tmp/mnn_models/



Note: I used this replacement for TranslateGemma-4B's chat template for Minja support and to not require extra inputs:
        "chat_template": "{{ bos_token }}\n{%- if (messages[0][\'role\'] != \'user\') -%}\n    {{ raise_exception("Conversations must start with a user prompt.") }}\n{%- endif -%}\n{%- for message in messages -%}\n    {%- if (message[\'role\'] == \'user\') != (loop.index0 % 2 == 0) -%}\n        {{ raise_exception("Conversation roles must alternate user/assistant/user/assistant/...") }}\n    {%- endif -%}\n    {%- if (message[\'role\'] == \'assistant\') -%}\n        {%- if message[\'content\'] is none or message[\'content\'] is not string -%}\n            {{ raise_exception("Assistant role must provide content as a string") }}\n        {%- endif -%}\n        {{ \'<start_of_turn>model\n\'}}\n        {{ message["content"] | trim }}\n    {%- elif (message[\'role\'] == \'user\') -%}\n        {{ \'<start_of_turn>user\n\' }}\n        {%- if message[\'content\'] is string -%}\n            {{ message["content"] | trim }}\n        {%- elif message[\'content\'] is iterable -%}\n            {%- set content = message["content"][0] -%}\n            {%- if content["type"] == \'text\' -%}\n                {{ content["text"] | trim }}\n            {%- elif content["type"] == \'image\' -%}\n                {{ \'<img>\' }}\n            {%- else -%}\n                {{ raise_exception("User role `content` can contain either `text` or `image` data.") }}\n            {%- endif -%}\n        {%- else -%}\n            {{ raise_exception("User role must provide content as a string or iterable.") }}\n        {%- endif -%}\n    {%- else -%}\n        {{ raise_exception("Conversations must only contain user or assistant roles.") }}\n    {%- endif -%}\n    {{ \'<end_of_turn>\n\' }}\n{%- endfor -%}\n{%- if add_generation_prompt -%}\n    {{\'<start_of_turn>model\n\'}}\n{%- endif -%}\n",
